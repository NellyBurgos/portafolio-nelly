📘 Proyecto destacado: Recordatorio Estudiantil

Aplicación web diseñada para ayudar a estudiantes a gestionar sus eventos académicos — parciales, entregas, exámenes — de forma clara, responsiva y accesible desde cualquier dispositivo.

🧩 Rol en el proyecto

Desarrollé toda la lógica de funcionamiento con JavaScript , estructuré el HTML para mantener una semántica clara y diseñé una interfaz adaptable a dispositivos móviles. Además, implementé funciones como listado dinámico, limpieza de eventos y control de entradas.

📌 Funcionalidades principales

  • Selección de materia y tipo de evento (parcial, final, entrega, etc.)
  • Asignación de fecha y hora
  • Botón para agregar evento a la lista
  • Botón para limpiar todos los eventos
  • Listado de eventos cargados (puede integrarse con localStorage)
  • Interfaz clara y responsiva para celulares
  • Tema claro/oscuro

🛠️ Tecnologías utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Versión futura: React + Firebase
